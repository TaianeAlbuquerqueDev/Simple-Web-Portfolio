# 🌐 Simple Web

Projeto de mockup de landing page desenvolvido com **HTML5** e **CSS3** puro. Criado com foco em estrutura limpa, layout moderno e boas práticas de marcação semântica.

---

## ✨ Seções da Página

- **Header** — Navegação com logo e links externos
- **Hero** — Seção de boas-vindas com título e subtítulo
- **Cards** — Três cards com ícone, título e descrição
- **Footer** — Rodapé com copyright

- Efeitos de hover com CSS
- Layout de cards com Flexbox/Grid
- Uso de ícones externos via CDN
